# Network Infrastructure & Design Labs | Cisco Packet Tracer

## 📌 Overview
This repository contains a series of networking labs developed during my Cybersecurity studies at City College Dublin. These projects focus on designing, configuring, and troubleshooting scalable and secure network infrastructures.

## 🛠️ Key Skills Demonstrated
* **Layer 2 Switching:** VLANs, Trunking (802.1Q), VTP, and Spanning Tree Protocol (STP).
* **Layer 3 Routing:** Inter-VLAN routing (Router-on-a-Stick), Static Routing, and Dynamic Routing (OSPF).
* **Network Services:** DHCP, DNS, and NAT configuration.
* **Security:** Port Security and Basic ACLs.

## 📂 Featured Labs

### 1. VLAN Segmentation & Inter-VLAN Routing
* **Goal:** Segregate department traffic (HR, IT, Sales) to improve security and broadcast control.
* **Key Techs:** Sub-interfaces, Trunking, VLAN tagging.

### 2. Spanning Tree Protocol (STP) Optimization
* **Goal:** Configure a redundant switch topology while preventing Layer 2 loops.
* **Key Techs:** Bridge Priority, Root Bridge selection, Rapid-PVST.
